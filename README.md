# StableMatch

A generic implementation of the Stable Match class of algorithms.

## Installation

### Without Bundler

Install it yourself:

```
$ gem install stable_match
```

### With Bundler

Add this line to your application's Gemfile:

```ruby
gem 'stable_match'
```

And then execute:

```
$ bundle
```
